Boot Up Kit for Fedora and IoT

Entry Profile
1. Students with ideas of circuits, preferably from engineering background
2. Students with in-depth knowledge of FOSS and Fedora 
3. Students with knowledge of  NodeJS or HTML preferred
4. Students with access to Raspberry Pis or Arduinos is a plus


Agenda
1. Introduction to IoT
2. Understanding the Architecture and Communication Flow in IoT
3. Understanding MQTT broker for beginners
4. Implementing a Node server with Johnny-Five and narf for advanced uses
5. Explanation of a few use cases 
6. Implementation and demonstration of one of use cases

Exit Profile
1. Having a clear idea of IoT and the ways to communicate 
2. Extending usecase to their own projects 
3. Uploading code to GitHub is a plus
