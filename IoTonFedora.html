Boot Up Kit for Fedora and IoT

Entry Profile
1. Students with idea of Circuts preferably from Engg. background
2. Students with indepth idea of FOSS and Fedora 
3. Students with knowledge of  node.js or html is preffered.
4. Students with access to raspberry pi or arduino is a plus


Agenda
1.Introduction to IoT
2.Understanding the Architecture and Communication Flow in IoT
3.Understanding MQTT broker for beginners
4.Implementing a node server with Jonny Five and narf for Advanced
5.Explaination of a few use cases 
6.Implementation and Demo of one of the use cases.

Exit Profile
1.Having a clear idea of IoT and the ways to communicate 
2.Extending usecase to their own projects 
3.Uploading code to github is a plus
