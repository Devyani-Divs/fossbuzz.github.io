BootUp Kit for Fedora and IoT

Entry Profile
1. Students with idea GitHub / Bitbucket
2. Students with in-depth idea of FOSS and Fedora 
3. Students with knowledge of  NodeJS or Angular preferred



Agenda
1. Introduction to application development
2. Difference between native applications and X platform applications
3. Using Angular 2/3 and Ionic to make simple X platform applications 
4. Using HTML5, CSS3, and JS to make lightweight web applications
5. Progressive hybrid app development
6. Implementation and demo of a few applications
7. Building Flask apps 


Exit Profile
1. Understanding of application development workflow
2. Extending usecase to their own projects 
3. Uploading code to GitHub is a plus
