Boot Up Kit for Fedora and IoT

Entry Profile
1. Students with idea Github/Bitbucket
2. Students with indepth idea of FOSS and Fedora 
3. Students with knowledge of  node.js or angular is preffered.



Agenda
1.Introduction to App Devlopment
2.Difference between Native App and X platform Apps
3.Using angular 2/3 and Ionic to make simple X platform apps 
4.Using html5 , CSS 3 and JS to make a light weight web app
5.Progressive hybrid app dev.
6.Implementation and demo of few apps
7.Building FLASK apps 


Exit Profile
1.Understanding of App Dev workflow
2.Extending usecase to their own projects 
3.Uploading code to github is a plus
