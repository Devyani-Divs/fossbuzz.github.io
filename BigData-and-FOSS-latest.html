BootUp Kits for Big Data and FOSS

Entry Profile
1. Students with an interest in data analytics
2. Experience with languages like Python and R preferred

Agenda
1. Introduction to big data and machine learning
2. Examples of ongoing work in different FOSS communities
3. Fedora and Big Data
4. How you can start contributing to interested teams 

Exit Profile
1. Having a clear idea of big data and the different related projects in FOSS
2. Having an idea of different ways to communicate
